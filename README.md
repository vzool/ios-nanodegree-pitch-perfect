# iOS Nanodegree Pitch Perfect

This is an iOS application build for recording audio from iPhone Microphone and then run some Sounds Effect on what you record.

Current Effects are:

- Pitch(High/Low)
- Speed Rate(High/Low)

# Requirements

- iOS 8.3+
- Xcode 6.3.2+
